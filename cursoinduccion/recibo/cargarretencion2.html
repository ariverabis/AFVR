Guía de Retenciones

Descripción de la Pantalla

Esta pantalla muestra una lista de las facturas para aplicarles una retención.

Funciones de la Tabla

  • Casilla de Selección: Use las casillas para seleccionar las facturas a las que desea aplicar la retención.
  • Monto (Bs): Es el valor de la retención a aplicar. Debe Moficarlo para que sea igual al comprobante que entrega el cliente. Para esto Pulse en el Monto.

Totales y Acciones.

  • Total (USD): Muestra el total de las retenciones seleccionadas.
  • Finalizar: Pulse este botón para guardar las retenciones asociadas a las facturas seleccionadas.
